# Production écrite en FLE et IA — site & kit pédagogique

Site présentant la soutenance de master 2 FLES d'Émilie Icyk, le mémoire
*« Production écrite en FLE et intelligence artificielle : un scénario pour une
autonomie assistée »*, et mettant en œuvre le kit pédagogique **« Production
écrite avec l'IA — 8 étapes clés pour gagner en autonomie rédactionnelle »**.

Site statique, accessible (WCAG 2.2 AA), sans base de données.
